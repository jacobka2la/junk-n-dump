# Y2K Lending

Official website for Y2K Lending, a private Hard Money Lending company focused on financing real estate investment opportunities.

## Website
https://y2klendingdone.vercel.app

## Technologies Used
- React
- Vite
- TypeScript
- Tailwind CSS

## About
Y2K Lending provides fast, flexible financing solutions for real estate investors. Our lending process prioritizes speed, transparency, and straightforward communication.

## Development
To run locally:

git clone https://github.com/jacobka2la/Y2KLending.git
cd Y2KLending
npm install
npm run dev

## Deployment
The site is deployed via Vercel.
