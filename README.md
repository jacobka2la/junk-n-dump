# Junk N Dump – Junk Removal Website

Official website for Junk N Dump, a locally owned junk removal business serving Oakland and Macomb County, Michigan.

## 🚛 About

Junk N Dump provides fast, reliable junk removal services for homeowners, renters, landlords, and contractors. We specialize in same-day cleanouts and straightforward pricing with free estimates.

## 🔧 Services Offered

- Furniture Removal
- Appliance Pickup
- Garage & Basement Cleanouts
- Full House Cleanouts
- Yard Debris Removal
- General Junk Hauling

## ⚡ Features

- Clean, modern UI
- Mobile responsive design
- Fast performance (powered by Vercel)
- Quote request form with image upload
- Simple navigation for customers
